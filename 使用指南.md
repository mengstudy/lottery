# 双色球数据分析系统 - 快速使用指南

## 🚀 快速开始

### 1. 启动应用
```bash
cd ssq_analyzer
python start.py
```

### 2. 访问网站
打开浏览器访问：http://localhost:5000

## 📋 主要功能

### 首页（最新数据）
- 显示最新开奖结果
- 遗漏值排行榜
- 冷热号分布
- 一键更新最新数据

### 历史数据
- 倒序显示（最新数据在第 1 页）
- 智能分页（最多显示 10 个页码）
- 支持页码跳转
- 红球不换行显示
- 中奖地区超长省略，鼠标悬浮显示全部

### 详情分析
- 点击期号查看完整分析
- 号码遗漏详情
- 奇偶比统计

## 🎯 核心特性

✅ **数据倒序** - 最新数据在第 1 页  
✅ **智能分页** - 精简页码显示，支持跳转  
✅ **立即保存** - 每页抓取后立即保存  
✅ **断点续传** - 中途中断不影响已保存数据  
✅ **地区显示** - 超长省略 + 悬浮提示  
✅ **红球排列** - 不换行，紧凑美观  

## 📊 数据更新

### 手动更新
1. 点击"更新最新数据"按钮 - 抓取最新 1 期
2. 点击"🔄 更新所有历史数据"按钮 - 抓取全部历史数据（约 3000 期）

### 自动更新
系统会在每周二、四、日 21:30（开奖时间）自动更新数据

## ️ 项目结构

```
ssq_analyzer/
├── app.py                  # Flask 主应用
├── start.py                # 启动脚本
├── requirements.txt        # 依赖列表
├── database/
│   └── db_manager.py       # 数据库管理
├── crawler/
│   └── crawler.py          # 数据抓取
├── analyzer/
│   └── stats_calculator.py # 统计分析
├── templates/
│   ├── index.html          # 首页
│   ├── history.html        # 历史数据（带分页跳转）
│   └── analysis.html       # 详情分析
└── data/
    └── lottery.db          # SQLite 数据库
```

## ⚙️ 技术栈

- **后端**: Flask 3.0 + SQLite
- **前端**: ECharts 5.4 + 响应式 CSS
- **爬虫**: requests + BeautifulSoup4
- **分析**: Pandas + collections

## 🔧 常见问题

**Q: 数据在哪里保存？**  
A: 数据库文件在 `data/lottery.db`

**Q: 如何查看日志？**  
A: 日志实时显示在终端窗口

**Q: 分页太多怎么办？**  
A: 使用底部的页码跳转功能，直接输入页码

**Q: 地区文字太长看不清？**  
A: 鼠标悬浮在文字上即可看到完整内容
