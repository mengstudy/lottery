# 更新日志 - 双色球数据分析系统

## v3.2.1 (2026-03-10) - 历史数据展示优化

### 🎯 主要改进

#### 1. 修复遗漏值计算逻辑
**问题**：历史列表和详情页的红球遗漏值不一致  
**原因**：数据排序错误导致遗漏值计算基准不对  
**解决**：
- 先复制升序数据用于计算遗漏值
- 再倒序原数据用于页面显示
- 通过索引转换确保数据正确对应

**影响**：
- ✅ 历史列表显示的遗漏值 = 详情页显示的遗漏值
- ✅ 所有期的遗漏值都正确
- ✅ 分页后数据仍然正确

---

#### 2. 红球遗漏详情显示优化
**旧版**：带彩色背景的徽章样式 `[0] [2] [7] [1] [9] [15]`  
**新版**：纯数字 + 短横线分隔 `0-1-2-7-9-15`  

**优势**：
- 简洁明了，一眼看到所有遗漏值的分布
- 节省空间，不再占用大量横向空间
- 易于对比，排序后更容易发现规律
- 视觉清爽，去掉花哨的样式，回归数据本身

**技术实现**：
```html+jinja
{% set sorted_missing = result.red_balls_missing|sort %}
{% for missing in sorted_missing %}
<span style="font-size: 13px; color: #333;">{{ missing }}</span>
{% if not loop.last %}
<span style="font-size: 13px; color: #999;">-</span>
{% endif %}
{% endfor %}
```

---

#### 3. 中奖地区悬浮显示增强
**旧版**：只有超过 50 个字符才显示 tooltip  
**新版**：所有地区都支持鼠标悬浮显示

**优化细节**：
- 列宽从 400px 缩小到 150px，节省空间
- 悬浮框最大宽度从 500px 增大到 600px
- 背景透明度从 0.9 加深到 0.95
- 动画效果从 linear 改为 ease-in-out（更平滑）
- 新增 `pointer-events: none` 防止遮挡鼠标事件

**CSS 改进**：
```css
.region-text {
    max-width: 150px;  /* 从 400px 减小 */
    overflow: hidden;
    text-overflow: ellipsis;
    white-space: nowrap;
}

.region-tooltip-text {
    max-width: 600px;  /* 从 500px 增大 */
    min-width: 200px;  /* 新增 */
    background-color: rgba(0, 0, 0, 0.95);  /* 从 0.9 加深 */
    transition: opacity 0.3s ease-in-out;   /* 更平滑 */
    pointer-events: none;  /* 新增 */
}
```
