# 彩票数据分析系统 - 使用指南

## 📖 系统简介

本系统是一个基于 Flask 的彩票数据分析网站，支持**双色球**和**大乐透**两种彩票分析：

- **双色球**：6 个红球（1-33）+ 1 个蓝球（1-16）
- **大乐透**：5 个前区红球（1-35）+ 2 个后区蓝球（1-12）

主要功能包括开奖数据查询、遗漏值统计、冷热号分析、数据可视化等。数据来源于官方 API，确保准确性和及时性。

## 🚀 快速开始

### 环境要求

- Python 3.8+
- Windows / macOS / Linux

### 安装依赖

```bash
pip install -r requirements.txt
```

### 启动应用

```bash
python start.py
```

### 访问网站

打开浏览器访问：**http://localhost:5000**

| 功能 | 地址 |
|------|------|
| 双色球首页 | http://localhost:5000 |
| 大乐透首页 | http://localhost:5000/dlt |

## 📋 功能模块

### 🅰️ 双色球

#### 1️⃣ 首页（双色球）

**功能特点：**
- 显示最新一期开奖结果
- 实时遗漏值排行榜（Top 10）
- 冷热号分布统计
- 一键更新最新数据
- 计算遗漏次数表

#### 2️⃣ 历史数据（双色球）

**功能特点：**
- 倒序排列：最新数据在第 1 页
- 智能分页：最多显示 10 个页码
- 页码跳转：直接输入页码快速定位

**主要操作：**
- **更新最新数据**：抓取最新 1 期开奖结果（需口令验证）
- **🔄 更新所有历史数据**：批量抓取所有历史数据（需口令验证）

#### 3️⃣ 详情分析（双色球）

**功能特点：**
- 完整的开奖信息展示
- 上一期/下一期遗漏分组对比
- 本期开出号码遗漏次数分组
- 冷热号分析

---

### 🅱️ 大乐透

#### 1️⃣ 首页（大乐透）

**访问地址：** http://localhost:5000/dlt

**功能特点：**
- 显示最新一期开奖结果（前区 5 红 + 后区 2 蓝）
- 前区遗漏值排行榜（Top 10）
- 后区遗漏值排行榜（Top 10）
- 冷热号分布统计

#### 2️⃣ 历史数据（大乐透）

**访问地址：** http://localhost:5000/dlt/history

**功能特点：**
- 前区 + 后区开奖号码展示
- 智能分页浏览
- 遗漏详情展示

#### 3️⃣ 详情分析（大乐透）

**访问地址：** http://localhost:5000/dlt/analysis/期号

**功能特点：**
- 上一期/下一期遗漏分组对比
- 本期开出号码遗漏次数分组
- 冷热号分析

#### 4️⃣ 历史遗漏分组（大乐透）

**访问地址：** http://localhost:5000/dlt/missing-groups

**功能特点：**
- 按期次展示遗漏分组详情
- 全部 35 个前区红球的遗漏分布
- 红色边框标记本期开出号码

## 🎯 遗漏次数分组说明

| 分组 | 遗漏次数 | 颜色 | 说明 |
|------|---------|------|------|
| A 组 | 0 次 | 🔴 红色 | 本期开出（热号） |
| B 组 | 1-5 次 | 🟡 黄色 | 温热号码 |
| C 组 | 6-8 次 | 🔵 蓝色 | 较冷号码 |
| D 组 | ≥9 次 | 🟢 青色 | 极冷号码 |

## 📊 数据更新

### 手动更新

#### 更新双色球数据
1. 访问双色球首页
2. 点击 **"立即更新"** 或 **"更新最新数据"**
3. 输入口令确认

#### 更新大乐透数据
1. 访问大乐透首页（/dlt）
2. 点击 **"立即更新"** 或在历史页面点击 **"更新最新数据"**
3. 输入口令确认

#### 更新所有历史数据
1. 访问历史数据页面
2. 点击 **"🔄 更新所有历史数据"**
3. 确认后等待抓取完成

## 🔧 常见问题

### Q1: 如何查看大乐透数据？
**A:** 访问 http://localhost:5000/dlt 或点击首页的"大乐透"按钮。

### Q2: 双色球和大乐透数据是分开的吗？
**A:** 是的，分别存储在 `lottery.db` 和 `dlt_lottery.db` 中。

### Q3: 如何只更新大乐透数据？
**A:** 在大乐透页面点击"立即更新"按钮。

### Q4: 大乐透的遗漏分组和双色球一样吗？
**A:** 类似，但大乐透前区有 35 个红球（双色球 33 个），分组方式相同。
